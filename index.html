Hands Paint
  • Draw on screen using hand gestures
  • Thumb + Index finger -> black dots
  • Thumb + Middle finger -> white dots
  • Thumb + Ring finger -> red dots
  • Thumb + Pinky finger -> clear screen
Hands VisJS
  • Thumb + Index finger (single hand) -> grab elements
  • Thumb + Index finger (both hands) -> zoom
  • Thumb + Middle finger -> trigger node function
  • Thumb + Ring finger -> trigger fountain at current position
Wormhole
  • rotate head to navigate through wormhole
  • control speed with your smile
  • open mouth or click to shoot
  • move pointer to aim
  • score points: remain in wormhole, hit targets
  • minimize config window and enable full-screen for best experience
